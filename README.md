# TextPrivate

TextPrivate is a secure, encrypted messaging Android application built with Jetpack Compose. It features a Secret Chat interface to disguise your communication and keep your messages safe.

## Features

- **Encrypted Sessions:** Messages are securely encrypted.
- **Disguise Mode:** Toggle an alternative UI to hide the chat from prying eyes.
- **Modern UI:** Built using Material Design 3 and Jetpack Compose.

## Architecture

- Built using modern Android architecture guidelines.
- Jetpack Compose for UI.
- MVVM (Model-View-ViewModel) architecture.
- Hilt for Dependency Injection.
